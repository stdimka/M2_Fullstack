DOM (Document Object Model) и BOM (Browser Object Model)

DOM определение

DOM — это объектная модель документа, представляющая HTML-страницу в виде дерева узлов, которые можно изменять с помощью JavaScript.

Узлы и элементы в DOM

DOM состоит из узлов (nodes), которые можно разделить на несколько типов:

  • Элементы (Element nodes) – узлы, представляющие HTML-теги.
  • Текстовые узлы (Text nodes) – узлы, содержащие текст внутри HTML-элементов.
  • Узлы комментариев (Comment nodes) – узлы, представляющие комментарии в коде (<!-- comment -->).
  • Документ (Document node) – корневой узел, представляющий весь HTML-документ (document).

Классификация узлов и элементов с примерами

Пример HTML-страницы:

<!DOCTYPE html>
<html>
<head>
    <title>Пример DOM</title>
</head>
<body>
    <h1>Заголовок</h1>
    <p>Текстовый узел <strong>с вложенным элементом</strong>.</p>
</body>
</html>

Структура DOM-дерева

document
 ├── html
 │    ├── head
 │    │    └── title ("Пример DOM")
 │    ├── body
 │         ├── h1 ("Заголовок")
 │         ├── p
 │              ├── (Text "Текстовый узел")
 │              └── strong ("с вложенным элементом")

Внешнее сходство с файловой системой

/ (Корень)
 ├── /html
 │    ├── /head
 │    │    ├── title.txt
 │    ├── /body
 │         ├── h1.txt
 │         ├── /p
 │              ├── text.txt
 │              └── strong.txt

BOM (Browser Object Model)

Определение

BOM — это объектная модель браузера, предоставляющая доступ к функционалу окна браузера, вкладок, истории и других возможностей, не связанных с HTML-документом.

Связь между BOM и DOM

  • BOM включает в себя DOM.
  • window — это главный объект BOM, который содержит:
    • document (DOM)
    • history, location, navigator, screen, console и другие объекты

Пример связи:

console.log(window.document.title); // Получаем заголовок страницы (из DOM)
console.log(window.location.href);  // Получаем текущий URL (из BOM)

Таким образом, DOM — это часть BOM, а BOM управляет поведением браузера в целом.
