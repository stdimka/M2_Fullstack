Методы доступа к элементам DOM

1. getElementById()

Метод используется для поиска элемента по его уникальному идентификатору (id).

Пример:


<div id="example">Hello, World!</div>
<script>
    let element = document.getElementById('example');
    console.log(element.innerText); // Выведет: Hello, World!
</script>

2. getElementsByClassName()

Этот метод находит все элементы с заданным классом и возвращает коллекцию (HTMLCollection).

Пример:


<div class="test">Item 1</div>
<div class="test">Item 2</div>
<script>
    let elements = document.getElementsByClassName('test');
    console.log(elements.length); // Выведет: 2
</script>

3. getElementsByTagName()>

Метод находит все элементы с указанным тегом (например, div, p, a и т.д.).

Пример:


<div>Item 1</div>
<p>Item 2</p>
<div>Item 3</div>
<script>
    let divs = document.getElementsByTagName('div');
    console.log(divs.length); // Выведет: 2
</script>

4. querySelector()

Метод находит первый элемент, который соответствует указанному CSS-селектору. Очень гибкий и мощный метод, позволяющий использовать любые CSS-селекторы.

Пример:


<div class="example">Hello</div>
<p class="example">World</p>
<script>
    let element = document.querySelector('.example');
    console.log(element.innerText); // Выведет: Hello
</script>

5. querySelectorAll()

Этот метод находит все элементы, соответствующие указанному CSS-селектору, и возвращает их в виде статической коллекции (NodeList).

Пример:


<div class="test">1</div>
<p class="test">2</p>
<div class="test">3</div>
<script>
    let elements = document.querySelectorAll('.test');
    console.log(elements.length); <!-- Выведет: 3 -->
</script>

6. getAttribute()

Метод используется для получения значения атрибута элемента.

Пример:


<a href="https://example.com" id="link">Example</a>
<script>
    let link = document.getElementById('link');
    console.log(link.getAttribute('href')); // Выведет: https://example.com
</script>

7. setAttribute()

Метод используется для изменения или добавления атрибута элемента.

Пример:


<button id="myButton">Click me</button>
<script>
    let button = document.getElementById('myButton');
    button.setAttribute('disabled', 'true'); // Добавит атрибут disabled
    console.log(button.getAttribute('disabled')); // Выведет: true
</script>

8. removeAttribute()

Метод используется для удаления атрибута у элемента.

Пример:


<button id="myButton" disabled>Click me</button>
<script>
    let button = document.getElementById('myButton');
    button.removeAttribute('disabled');
    console.log(button.hasAttribute('disabled')); // Выведет: false
</script>

9. innerHTML

Этот метод используется для получения или установки HTML-содержимого внутри элемента.

Пример:


<div id="container">Old content</div>
<script>
    let container = document.getElementById('container');
    console.log(container.innerHTML); // Выведет: Old content
    container.innerHTML = 'New content';
    console.log(container.innerHTML); // Выведет: New content
</script>

10. textContent

Этот метод аналогичен innerHTML, но работает только с текстом, игнорируя HTML-теги внутри элемента.

Пример:


<div id="container">Hello <strong>World</strong></div>
<script>
    let container = document.getElementById('container');
    console.log(container.textContent); // Выведет: Hello World
</script>

11. parentNode

Метод возвращает родительский элемент для указанного элемента.

Пример:


<div id="parent">
    <p id="child">Child element</p>
</div>
<script>
    let child = document.getElementById('child');
    let parent = child.parentNode;
    console.log(parent.id); // Выведет: parent
</script>

12. childNodes

Этот метод возвращает все дочерние узлы элемента в виде коллекции NodeList.

Пример:


<div id="container">
    <p>Paragraph 1</p>
    <p>Paragraph 2</p>
</div>
<script>
    let container = document.getElementById('container');
    let children = container.childNodes;
    console.log(children.length); // Выведет: 3 (включая текстовые узлы)
</script>

13. firstChild и lastChild

Эти методы возвращают первый и последний дочерний узел элемента.

Пример:


<div id="container">
    <p>First paragraph</p>
    <p>Second paragraph</p>
</div>
<script>
    let container = document.getElementById('container');
    console.log(container.firstChild); // Выведет: текстовый узел (перед <p>)
    console.log(container.lastChild); // Выведет: текстовый узел (после <p>)
</script>

14. nextSibling и previousSibling

Эти методы возвращают следующий и предыдущий узлы на одном уровне с текущим.

Пример:


<div id="container">
    <p>Paragraph 1</p>
    <p>Paragraph 2</p>
</div>
<script>
    let firstParagraph = document.querySelector('p');
    console.log(firstParagraph.nextSibling); // Выведет: второй <p>
    console.log(firstParagraph.previousSibling);
                            // Выведет: текстовый узел (перед <p>)
</script>
