Назад

Методы навигации по DOM-узлам

1. Методы и свойства для работы с родительскими узлами:

parentNode

Возвращает родительский узел для текущего элемента.

Пример:


const element = document.querySelector('div');
const parent = element.parentNode;
console.log(parent); // Выведет родительский узел

2. Методы и свойства для работы с дочерними узлами:

Здесь прежде всего необходимо отметить разницу дочерним узлом (child node) и элементным узлом (element node):

Дочерний узел (child node) — это любой узел, находящийся внутри родительского узла. Он может быть:

  • Элементом (ELEMENT_NODE) — <div>, <p>, <a> и т. д.
  • Текстовым узлом (TEXT_NODE) — текст между тегами.
  • Комментарием (COMMENT_NODE) — <!-- комментарий -->.
  • Другими типами узлов (например, DocumentFragment).

Элементный узел (element node) — это только узел, который является HTML- или SVG-элементом (<div>, <p>, <a> и т. д.). Он не включает текстовые узлы и комментарии.

Пример: Разница между childNodes и children


<div id="parent">
    Текстовый узел
    <p>Параграф</p>
    <!-- Комментарий -->
    <span>Спан</span>
</div>
        

const parent = document.getElementById("parent");

console.log(parent.childNodes);
// NodeList(5) [text, p, text, comment, span]

console.log(parent.children);
// HTMLCollection(2) [p, span]
  • childNodes содержит все узлы, включая текст и комментарии.
  • children содержит только элементные узлы (<p> и <span>).

childNodes

Возвращает коллекцию всех дочерних узлов, включая текстовые узлы и комментарии.

Пример:


const parent = document.querySelector('div');
const children = parent.childNodes;
console.log(children); // Массив всех дочерних узлов,
                       // включая текстовые и комментарии

firstChild

Возвращает первый дочерний узел (включая текстовые и комментарии).

Пример:


const parent = document.querySelector('div');
const firstChild = parent.firstChild;
console.log(firstChild); // Первый дочерний узел (может быть текстом)

lastChild

Возвращает последний дочерний узел (включая текстовые и комментарии).

Пример:


const parent = document.querySelector('div');
const lastChild = parent.lastChild;
console.log(lastChild); // Последний дочерний узел

3. Методы и свойства для работы с элементами-соседями:

nextElementSibling

Возвращает следующий элементный узел (игнорирует текстовые узлы и комментарии).

Пример:


const element = document.querySelector('div');
const nextSibling = element.nextElementSibling;
console.log(nextSibling); // Следующий элементный узел

previousElementSibling

Возвращает предыдущий элементный узел (игнорирует текстовые узлы и комментарии).

Пример:


const element = document.querySelector('div');
const prevSibling = element.previousElementSibling;
console.log(prevSibling); // Предыдущий элементный узел

4. Методы и свойства для работы с первым и последним элементом:

firstElementChild

Возвращает первый дочерний элементный узел (игнорирует текстовые узлы и комментарии).

Пример:


const parent = document.querySelector('div');
const firstElementChild = parent.firstElementChild;
console.log(firstElementChild); // Первый дочерний элементный узел

lastElementChild

Возвращает последний дочерний элементный узел (игнорирует текстовые узлы и комментарии).

Пример:


const parent = document.querySelector('div');
const lastElementChild = parent.lastElementChild;
console.log(lastElementChild); // Последний дочерний элементный узел

Пример в одном блоке:

Предположим, у нас есть HTML-структура:


<div id="parent">
    <p>Первый абзац</p>
    <p>Второй абзац</p>
    <span>Текст в спане</span>
    <p>Третий абзац</p>
</div>

Пример использования навигации по DOM:


const parent = document.getElementById('parent');

// Получаем родительский узел
console.log(parent.parentNode); // null, так как родитель для <div> нет

// Получаем все дочерние узлы
console.log(parent.childNodes); // Включая текстовые узлы и комментарии

// Получаем первый дочерний элементный узел
console.log(parent.firstElementChild); // <p>Первый абзац</p>

// Получаем последний дочерний элементный узел
console.log(parent.lastElementChild); // <p>Третий абзац</p>

// Получаем следующий элементный узел
const firstParagraph = parent.firstElementChild;
console.log(firstParagraph.nextElementSibling); // <p>Второй абзац</p>

// Получаем предыдущий элементный узел
console.log(firstParagraph.nextElementSibling.previousElementSibling); // <p>Первый абзац<</p>

Заключение:

Эти методы и свойства позволяют гибко манипулировать DOM, обеспечивая доступ к родительским и дочерним узлам, а также к соседним элементам. Вы можете комбинировать их для выполнения сложных операций с деревом элементов на странице.
