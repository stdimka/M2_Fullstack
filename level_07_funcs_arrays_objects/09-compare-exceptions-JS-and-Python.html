Сравнение исключений в JS и Python

Характеристика JavaScript Python
Базовые исключения Error, TypeError, ReferenceError, SyntaxError, RangeError и другие Exception, TypeError, ReferenceError, SyntaxError, ValueError и другие
Создание исключений throw new Error('Message') raise Exception('Message')
Обработка исключений try { ... } catch (e) { ... } finally { ... } try: ... except Exception as e: ... finally: ...
Перехват исключений В блоке catch можно указать переменную (например, catch (e)) для доступа В блоке except можно указать переменную (например, except Exception as e)
Множественные перехваты Один catch для нескольких типов ошибок через проверку типа (instanceof) Можно указать несколько except для разных типов ошибок
Собственные исключения Создаются через классы, например: class MyError extends Error { ... } Создаются через классы, например: class MyError(Exception): ...
Печать стека ошибки console.error(e) или e.stack print(e) или traceback.print_exc()
Обработка конкретных ошибок Для обработки конкретных типов ошибок используются операторы типа instanceof Для обработки конкретных типов ошибок используется указание типа в except
Перехват ошибок асинхронно В промисах используется .catch() Используется try/except в асинхронных функциях с async/await
