Явное преобразование

Явное преобразование выполняется вручную с использованием специальных функций.

Метод Описание Пример
String(value) Преобразует значение в строку. 
String(123) → "123"
Number(value) Преобразует значение в число. 
Number("42") → 42
Boolean(value) Преобразует значение в логический тип. 
Boolean(1) → true
parseInt(value, radix)
где radix - основание системы счисления (опционально) 		Преобразует строку в целое число. 
parseInt("10.5") → 10

parseInt("1010", 2) → 10
parseFloat(value) Преобразует строку в число с плавающей точкой. 
parseFloat("10.5") → 10.5
toString() Преобразует число или объект в строку. 
(123).toString() → "123"

Неявное преобразование

Неявное (автоматическое) преобразование выполняется JavaScript при необходимости.

Выражение Описание Результат
"5" + 3 Число преобразуется в строку (конкатенация). 
"53"
"5" - 3 Строка преобразуется в число. 
2
"10" * "2" Обе строки преобразуются в числа. 
20
true + false Булевы значения преобразуются в числа (1 и 0). 
1
null + 1 null преобразуется в 0. 
1
undefined + 1 undefined преобразуется в NaN. 
NaN
"5" == 5 Строка "5" неявно преобразуется в число. 
true
"5" === 5 Строгое сравнение без преобразования. 
false

Дополнительные правила преобразования

Использование знака + перед строкой

Добавление `+` перед строкой, содержащей число, преобразует её в число.

let str = "42";
let num = +str;
console.log(num); // 42 (число)

Правило арифметических операций

Все арифметические операции (кроме `+`) приводят операнды к числовому типу.

console.log("10" - 2);  // 8
console.log("10" * "2"); // 20
console.log("10" / 2);   // 5
console.log("10" + 2);   // "102"
