Создание объектов в JavaScript

В JavaScript объект — это структура данных, которая позволяет хранить набор свойств в виде пар "ключ-значение". Объекты используются для представления сущностей с их характеристиками и поведением. Существует несколько способов создания объектов в JavaScript. Вот основные из них:

1. Литерал объекта


const user1 = {
    name: "Иван",
    age: 30
};
console.log("Литерал объекта:", user1);

2. Функция-конструктор


function User(name, age) {
    this.name = name;
    this.age = age;
}
const user2 = new User("Мария", 25);
console.log("Функция-конструктор:", user2);

3. Классы (ES6)


class UserClass {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
}
const user3 = new UserClass("Алексей", 35);
console.log("Класс:", user3);

4. Object.create


const person = {
    greet() {
        console.log("Привет!");
    }
};
const user4 = Object.create(person);
user4.name = "Ольга";
console.log("Object.create:", user4);

5. Object.assign

Этот метод копирует свойства из одного или нескольких исходных объектов в целевой объект. Его удобно использовать для создания новых объектов на основе существующих. 


const user5 = Object.assign({}, { name: "Дмитрий", age: 40 });
console.log("Object.assign:", user5);

6. new Object()


const user6 = new Object();
user6.name = "Сергей";
user6.age = 28;
console.log("new Object():", user6);

7. Фабричная функция

Фабрика объектов — это функция, которая возвращает новый объект без явного использования ключевого слова new. Это шаблон проектирования, позволяющий создавать объекты с заданной структурой. 


function createUser(name, age) {
    return { name, age };
}
const user7 = createUser("Елена", 22);
console.log("Фабричная функция:", user7);
