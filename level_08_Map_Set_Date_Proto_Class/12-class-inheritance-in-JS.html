Наследование в JavaScript при создании классов

1. Базовые концепции наследования

В JavaScript классы поддерживают наследование с помощью ключевого слова extends. Это позволяет одному классу (дочернему) наследовать свойства и методы другого класса (родительского). 


class Animal {
  constructor(name) {
    this.name = name;
  }

  makeSound() {
    console.log("Звук животного");
  }
}

class Dog extends Animal {
  bark() {
    console.log("Гав-гав");
  }
}

const dog = new Dog("Шарик");
console.log(dog.name); // Шарик
dog.makeSound(); // Звук животного
dog.bark(); // Гав-гав

2. Использование super в конструкторах

Ключевое слово super позволяет вызвать конструктор родительского класса, чтобы установить свойства, объявленные в нем. 


class Animal {
  constructor(name) {
    this.name = name;
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name); // Вызываем конструктор родителя
    this.breed = breed;
  }
}

const dog = new Dog("Шарик", "Овчарка");
console.log(dog.name); // Шарик
console.log(dog.breed); // Овчарка

3. Переопределение методов и использование super

Можно переопределять методы родительского класса, но при этом использовать super для вызова оригинального метода. 


class Animal {
  makeSound() {
    console.log("Какой-то звук...");
  }
}

class Dog extends Animal {
  makeSound() {
    super.makeSound(); // Вызываем родительский метод
    console.log("Гав-гав");
  }
}

const dog = new Dog();
dog.makeSound();
// Какой-то звук...
// Гав-гав

4. Наследование от встроенных классов

Можно наследоваться от встроенных классов, таких как Array, Error и HTMLElement. 


class MyArray extends Array {
  last() {
    return this[this.length - 1];
  }
}

const arr = new MyArray(1, 2, 3);
console.log(arr.last()); // 3

5. Статические методы и наследование

Статические методы наследуются, но могут быть переопределены.


class Parent {
  static sayHello() {
    console.log("Привет от родителя!");
  }
}

class Child extends Parent {
  static sayHello() {
    super.sayHello();
    console.log("Привет от ребенка!");
  }
}

Child.sayHello();
// Привет от родителя!
// Привет от ребенка!
