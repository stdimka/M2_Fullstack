Конструкторы и методы классов в JavaScript

1. Конструкторы классов

В классе может быть только один конструктор.

Явный конструктор

Когда мы определяем конструктор, он вызывается при создании объекта:


class Person {
  constructor(name, age) { // Явный конструктор
    this.name = name;
    this.age = age;
  }
}

const person = new Person("Иван", 30);
console.log(person.name); // Иван
console.log(person.age);  // 30

Конструктор по умолчанию

Если мы не определяем конструктор, он создаётся автоматически и ничего не делает: 


class Animal {}

const cat = new Animal(); // Конструктор по умолчанию
console.log(cat); // Animal {}

2. Методы классов

Методы в классах бывают обычные, приватные и статические.

Обычные методы

Они доступны у всех экземпляров класса.


class User {
  constructor(name) {
    this.name = name;
  }

  sayHello() { // Обычный метод
    return `Привет, меня зовут ${this.name}`;
  }
}

const user = new User("Олег");
console.log(user.sayHello()); // Привет, меня зовут Олег

Приватные методы (#)

Приватные методы можно вызывать только внутри класса.


class BankAccount {
  #balance = 0; // Приватное свойство

  constructor(owner) {
    this.owner = owner;
  }

  deposit(amount) {
    this.#balance += amount;
    console.log(`Баланс: ${this.#balance}`);
  }

  #getBalance() { // Приватный метод
    return this.#balance;
  }
}

const account = new BankAccount("Алиса");
account.deposit(100);
// console.log(account.#getBalance()); // Ошибка! Приватный метод

Статические методы (static)

Статические методы принадлежат классу, а не объекту.


class MathHelper {
  static sum(a, b) { // Статический метод
    return a + b;
  }
}

console.log(MathHelper.sum(2, 3)); // 5
// console.log(new MathHelper().sum(2, 3)); // Ошибка! sum не у экземпляра

Выводы:

  1. Конструктор
    • Явный — определён внутри класса (constructor).
    • По умолчанию — если не определён, создаётся пустой.
  2. Методы
    • Обычные — methodName() {...}.
    • Приватные — #methodName() {...} (доступны только внутри класса).
    • Статические — static methodName() {...} (вызываются через класс).
