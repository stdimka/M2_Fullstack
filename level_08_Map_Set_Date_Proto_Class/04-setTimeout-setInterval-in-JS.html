Сравнение setTimeout и setInterval

В JavaScript есть два метода для выполнения кода с задержкой или через определенные интервалы: setTimeout и setInterval. Рассмотрим их различия.

Основные различия

Метод Описание
setTimeout Выполняет переданную функцию один раз через указанное время (в миллисекундах).
setInterval Выполняет переданную функцию многократно через заданные промежутки времени.

Очистка таймеров

Если вам нужно отменить выполнение, используйте:

  • clearTimeout(id) — отменяет выполнение setTimeout, если оно еще не произошло.
  • clearInterval(id) — останавливает выполнение setInterval.

Примеры кода

Использование setTimeout


setTimeout(() => {
    console.log("Выполнится через 3 секунды");
}, 3000);

Этот код выведет сообщение в консоль через 3 секунды после запуска.

Использование setInterval


const intervalId = setInterval(() => {
    console.log("Каждые 2 секунды");
}, 2000);

Этот код будет выводить сообщение в консоль каждые 2 секунды, пока не будет остановлен.

Остановка setInterval с clearInterval


const intervalId = setInterval(() => {
    console.log("Сообщение каждые 2 секунды");
}, 2000);

setTimeout(() => {
    clearInterval(intervalId);
    console.log("Интервал остановлен");
}, 6000);

Этот код остановит выполнение интервала через 6 секунд.
