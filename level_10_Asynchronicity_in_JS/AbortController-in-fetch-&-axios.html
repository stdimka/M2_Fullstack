AbortController в Fetch API и Axios

AbortController — это встроенный в JavaScript API, который используется для отмены асинхронных операций, таких как fetch. Он работает в связке с объектом AbortSignal, который передаётся в опции запроса и позволяет отменить выполнение при необходимости.

Как работает AbortController?

  • Создаётся контроллер — const controller = new AbortController().
  • Получаем сигнал — const signal = controller.signal.
  • Передаём сигнал в fetch — fetch(url, { signal }).
  • Вызываем controller.abort() — чтобы отменить запрос.

В обоих примерах запрос отменяется по таймеру (для случая очень долгих ответов сервера). Можно также сделать отмену по нажатию кнопки и т.д.
