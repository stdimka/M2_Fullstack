HTML-элемент <iframe>

Тег <iframe> используется для встраивания другого веб-документа в текущую страницу.

Основные атрибуты

  • src – указывает URL-адрес загружаемого ресурса.
  • width – задает ширину iframe.
  • height – задает высоту iframe.
  • frameborder – управляет отображением рамки (0 – убрать, 1 – показать).
  • allowfullscreen – разрешает полноэкранный режим (требуется дополнительный JS скрипт).
  • name – задает имя фрейма, используется при отправке форм.
  • referrerpolicy – определяет, передавать ли реферер (т.е. можно ли встроенному сайту передавать данные вашего ресурса).

Встроенная Google Карта

Обычная вставка url в источник фрейма iframe src="" даст ошибку. Необходимо заменить обычный url на embed-url:

  1. Открыть Google Maps → https://www.google.com/maps.
  2. Выбрать нужное место на карте.
  3. Нажать "Поделиться" → "Копировать HTML-код"
  4. Вставить полученный код в html страницу
