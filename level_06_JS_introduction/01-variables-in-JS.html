Типы данных

Переменные и типы данных в JavaScript

В JavaScript переменные используются для хранения данных. Объявить переменную можно с помощью var, let или const.

Объявление переменных: var, let, const

В JavaScript переменные можно объявлять тремя способами:

  • var – устаревший способ объявления переменной, имеет функциональную область видимости.
  • let – позволяет объявлять переменные с блочной областью видимости.
  • const – используется для объявления неизменяемых переменных с блочной областью видимости.

Области видимости

В JavaScript существует три типа областей видимости:

  • Глобальная область – переменные доступны во всем коде.
  • Функциональная область – переменные, объявленные с помощью var внутри функции, доступны только в этой функции.
  • Блочная область – переменные, объявленные с помощью let или const внутри блока {}, доступны только в этом блоке.

Hoisting (всплытие переменных и функций)

Hoisting – это механизм в JavaScript, при котором объявления переменных и функций поднимаются в начало своей области видимости до выполнения кода.

Фактически, это означает, что переменные, объявленные с помощью var, и функции можно использовать до их объявления в коде.

Однако переменные, объявленные через let и const, поднимаются, но остаются в "временной мертвой зоне" (TDZ) и не могут быть использованы до фактического объявления.

Пример hoisting:

console.log(a); // undefined (НЕ ошибка!)
var a = 5;
console.log(a); // 5

console.log(b); // ❌ ReferenceError
let b = 10;
console.log(b);

Сравнение var, let, const

Характеристика var let const
Область видимости Функциональная или глобальная Блочная Блочная
Hoisting (всплытие) Да, но инициализация не всплывает Нет (ошибка при обращении до объявления) Нет (ошибка при обращении до объявления)
Переназначение Да Да Нет
Инициализация (т.е. немедленное присвоение значения при объявлении) Необязательна Необязательна Обязательна

Примитивные типы данных и объекты (непримитивные типы данных)

В JavaScript существует два типа данных: примитивные и объекты (непримитивные).

Примитивные типы данных

  • String – строка, например "Привет, мир!"
  • Number – число, например 42 или 3.14
  • BigInt – большие числа, например 9007199254740991n
  • Boolean – логическое значение true или false
  • Undefined – переменная объявлена, но не имеет значения
  • Null – значение отсутствует
  • Symbol – уникальный идентификатор

Объекты (непримитивные типы данных)

  • Object – сложная структура данных, включая:
    • Обычный объект: let obj = { name: "Alex", age: 30 };
    • Array (массив): let arr = [1, 2, 3];
    • Function (функция): function sayHello() { console.log("Hello"); }
    • Date (дата): let date = new Date();
    • RegExp (регулярное выражение): let regex = /ab+c/;
    • Map: let map = new Map(); map.set('key', 'value');
    • Set: let set = new Set([1, 2, 3]);
    • WeakMap и WeakSet: Аналоги Map и Set, но с "слабыми" ссылками
    • Promise: let promise = new Promise((resolve) => resolve("done"));
    • Error: throw new Error("Something went wrong");

Примитивы vs. Объекты

  • Примитивы → ведут себя как immutable (неизменяемые) в Python.
  • Объекты → ведут себя как ссылочные типы (mutable) в Python.

Пример кода

Примитивные типы данных

let stringVar = "Привет, мир!";
let numberVar = 42;
let floatVar = 3.14;
let bigIntVar = 9007199254740991n;
let booleanVar = true;
let undefinedVar;
let nullVar = null;
let symbolVar = Symbol("id");

console.log(stringVar);
console.log(numberVar);
console.log(floatVar);
console.log(bigIntVar);
console.log(booleanVar);
console.log(undefinedVar);
console.log(nullVar);
console.log(symbolVar);

Непримитивные типы данных

let objectVar = { name: "Иван", age: 30 };
let arrayVar = [1, 2, 3];
let functionVar = function() { console.log("Функция"); };
let dateVar = new Date();
let regexVar = /ab+c/;
let mapVar = new Map();
mapVar.set("key", "value");
let setVar = new Set([1, 2, 3]);
let promiseVar = new Promise((resolve) => resolve("done"));
let errorVar = new Error("Ошибка");

console.log(objectVar);
console.log(arrayVar);
console.log(functionVar);
console.log(dateVar);
console.log(regexVar);
console.log(mapVar);
console.log(setVar);
console.log(promiseVar);
console.log(errorVar);
