Назад

Все виды ветвлений в JavaScript

1. Условный оператор if

Оператор if используется для выполнения кода, если условие истинно.

let age = 20;

if (age >= 18) {
    console.log("Вы совершеннолетний.");
}

В данном примере сообщение "Вы совершеннолетний." будет выведено, так как переменная age равна 20, и условие age >= 18 истинно.

2. Условный оператор if...else

Оператор if...else позволяет выполнить один блок кода, если условие истинно, и другой — если оно ложно.

let age = 16;

if (age >= 18) {
    console.log("Вы совершеннолетний.");
} else {
    console.log("Вы несовершеннолетний.");
}

Если значение переменной age меньше 18, то будет выведено сообщение "Вы несовершеннолетний."

3. Условный оператор if...else if...else

Оператор if...else if...else используется для проверки нескольких условий поочередно.

let age = 20;

if (age < 13) {
    console.log("Вы ребенок.");
} else if (age >= 13 && age < 18) {
    console.log("Вы подросток.");
} else {
    console.log("Вы взрослый.");
}

В этом примере для переменной age будет выведено сообщение "Вы взрослый.", так как age равно 20, и первое и второе условия не выполняются.

4. Тернарный оператор

Тернарный оператор — это короткая форма записи условного оператора. Он имеет следующий синтаксис: условие ? выражение1 : выражение2.

let age = 17;
let result = (age >= 18) ? "Вы совершеннолетний." : "Вы несовершеннолетний.";
console.log(result);

Если условие age >= 18 истинно, то будет выведено сообщение "Вы совершеннолетний.", в противном случае — "Вы несовершеннолетний."

5. Оператор switch

Оператор switch используется для проверки нескольких вариантов. Он позволяет сравнивать значение с несколькими возможными вариантами и выполнять соответствующие действия.

let fruit = "apple";

switch (fruit) {
    case "apple":
        console.log("Это яблоко.");
        break;
    case "banana":
        console.log("Это банан.");
        break;
    default:
        console.log("Неизвестный фрукт.");
}

В этом примере будет выведено сообщение "Это яблоко.", так как переменная fruit равна "apple". Оператор break используется для выхода из оператора switch после выполнения одного из условий.

6. Оператор switch с безусловным переходом

Вместо break можно использовать безусловный переход через return или "выход" из блока, если необходимо завершить выполнение после первого совпадения.

let fruit = "banana";

switch (fruit) {
    case "apple":
        console.log("Это яблоко.");
        return;
    case "banana":
        console.log("Это банан.");
        return;
    default:
        console.log("Неизвестный фрукт.");
}

В этом примере будет выведено "Это банан.", так как в switch нет необходимости продолжать выполнение после первого совпадения.
