Логические операторы и операторы сравнения

1. Логические операторы

1. Оператор "И" (&&)

Оператор && возвращает true, если оба операнда истинны. В противном случае возвращает false.

let a = true;
let b = false;
console.log(a && b); // false

console.log(5 > 2 && 8 > 3); // true

Если хотя бы одно из условий ложно, результат будет false.

2. Оператор "ИЛИ" (||)

Оператор || возвращает true, если хотя бы один операнд истинный. Если оба ложны, возвращает false.

let a = true;
let b = false;
console.log(a || b); // true

console.log(5 < 2 || 8 > 3); // true

Если хотя бы одно из условий истинно, результат будет true.

3. Оператор "НЕ" (!)

Оператор ! инвертирует логическое значение. Если значение true, оно становится false, и наоборот.

let a = true;
console.log(!a); // false

console.log(!(5 > 2)); // false

Этот оператор используется для проверки отрицания условий.

4. Оператор нулевого слияния (??)

Оператор ?? возвращает первый операнд, если он не равен null или undefined, иначе возвращает второй операнд.

let name = null;
let defaultName = "Гость";
console.log(name ?? defaultName); // "Гость"

let value = 0;
console.log(value ?? 10); // 0

Этот оператор полезен для установки значений по умолчанию.

2. Операторы сравнения

Оператор Описание Пример Результат
== (нестрогое равенство) Сравнивает значения без учёта типа 
console.log("5" == 5);
 true
=== (строгое равенство) Сравнивает значения и их тип 
console.log("5" === 5);
 false
!= (нестрогое неравенство) Сравнивает значения без учёта типа 
console.log("5" != 5);
 false
!== (строгое неравенство) Сравнивает значения и их тип 
console.log("5" !== 5);
 true
> (больше) Возвращает true, если первый операнд больше второго 
console.log(10 > 5);
 true
< (меньше) Возвращает true, если первый операнд меньше второго 
console.log(10 < 5);
 false
>= (больше или равно) Возвращает true, если первый операнд больше или равен второму 
console.log(5 >= 5);
 true
<= (меньше или равно) Возвращает true, если первый операнд меньше или равен второму 
console.log(3 <= 5);
 true

3. Приоритет операторов

Приоритет выполнения операторов (от высокого к низкому):

  • ! (НЕ)
  • >, <, >=, <= (сравнение)
  • ==, ===, !=, !== (равенство)
  • && (И)
  • || (ИЛИ)
  • ?? (Нулевое слияние)

4. Использование логических операторов в условиях

let age = 20;
if (age >= 18 && age <= 65) {
    console.log("Вы в трудоспособном возрасте.");
} else {
    console.log("Вы не в трудоспособном возрасте.");
}

Этот код проверяет, находится ли возраст в диапазоне от 18 до 65 лет.
