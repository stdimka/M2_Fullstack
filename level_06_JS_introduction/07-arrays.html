Массивы в JavaScript

1. Создание массива

Массивы в JavaScript можно создать несколькими способами:

// Использование квадратных скобок
let arr1 = [1, 2, 3, 4, 5];

// Использование конструктора new Array()
let arr2 = new Array(5); // Создаёт массив из 5 пустых элементов
let arr3 = new Array(1, 2, 3, 4, 5); // Создаёт массив с указанными значениями

2. Доступ к элементам массива

let numbers = [10, 20, 30, 40];
console.log(numbers[0]); // 10
console.log(numbers[2]); // 30

3. Изменение элементов массива

let fruits = ["Яблоко", "Банан"];
fruits[1] = "Апельсин"; // Замена элемента
console.log(fruits); // ["Яблоко", "Апельсин"]

4. Методы массивов

Метод Описание Пример Результат
push() Добавляет элемент в конец массива 
let arr = [1, 2];
arr.push(3);
 [1, 2, 3]
pop() Удаляет последний элемент массива 
let arr = [1, 2, 3];
arr.pop();
 [1, 2]
unshift() Добавляет элемент в начало массива 
let arr = [2, 3];
arr.unshift(1);
 [1, 2, 3]
shift() Удаляет первый элемент массива 
let arr = [1, 2, 3];
arr.shift();
 [2, 3]
concat() Объединяет два массива 
let arr = [1, 2];
let arr2 = arr.concat([3, 4]);
 [1, 2, 3, 4]
slice() Создаёт копию части массива 
let arr = [1, 2, 3];
let sub = arr.slice(1, 3);
 [2, 3]
splice() Удаляет или заменяет элементы 
let arr = [1, 2, 3];
arr.splice(1, 1, 99);
 [1, 99, 3]
forEach() Вызывает функцию для каждого элемента 
let arr = [1, 2, 3];
arr.forEach(n => console.log(n * 2));
 2, 4, 6
map() Создаёт новый массив, применяя функцию 
let arr = [1, 2, 3];
let newArr = arr.map(n => n * 2);
 [2, 4, 6]
filter() Фильтрует массив по условию 
let arr = [1, 2, 3, 4];
let even = arr.filter(n => n % 2 === 0);
 [2, 4]
find() Находит первый элемент, соответствующий условию 
let arr = [1, 2, 3];
let found = arr.find(n => n > 1);
 2
reduce() Сводит массив к одному значению 
let arr = [1, 2, 3];
let sum = arr.reduce((acc, n) => acc + n, 0);
 6

5. Проверка наличия элемента

let numbers = [10, 20, 30];
console.log(numbers.includes(20)); // true
console.log(numbers.indexOf(30)); // 2

6. Перебор массива

let fruits = ["Яблоко", "Банан", "Апельсин"];
for (let fruit of fruits) {
    console.log(fruit);
}
