Все виды циклов в JavaScript

1. Цикл for

Цикл for используется для повторения блока кода определённое количество раз. Он состоит из трёх частей: инициализация, условие и шаг.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Этот цикл будет выводить числа от 0 до 4. Он начинается с инициализации переменной i со значением 0, затем проверяет условие i < 5, и после каждого выполнения увеличивает i на 1.

2. Цикл while

Цикл while выполняет блок кода, пока условие истинно. Важно следить за тем, чтобы условие со временем становилось ложным, иначе цикл может стать бесконечным.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

В этом примере цикл продолжает выполняться, пока i меньше 5. После каждого выполнения i увеличивается на 1, что приведёт к завершению цикла, когда условие станет ложным.

3. Цикл do...while

Цикл do...while отличается от while тем, что блок кода выполняется хотя бы один раз, а затем проверяется условие.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Цикл выполнит тело хотя бы один раз, даже если условие i < 5 изначально ложное. В данном примере цикл выведет числа от 0 до 4.

4. Цикл for...in

Цикл for...in используется для перебора всех перечислимых свойств объекта или элементов массива по индексу. Он часто используется для объектов, но может быть использован и для массивов, хотя для массивов предпочтительнее использовать for...of.

let person = { name: "Иван", age: 30, city: "Москва" };

for (let key in person) {
    console.log(key + ": " + person[key]);
}

Этот цикл перебирает все свойства объекта person и выводит их имена и значения. В данном примере будут выведены: "name: Иван", "age: 30", "city: Москва".

5. Цикл for...of

Цикл for...of используется для перебора элементов массива или других итерируемых объектов (например, строк). Он позволяет работать с самими элементами, а не с их индексами.

let numbers = [1, 2, 3, 4, 5];

for (let num of numbers) {
    console.log(num);
}

Этот цикл перебирает элементы массива numbers и выводит их значения. В данном примере будут выведены числа от 1 до 5.

6. Цикл forEach

Метод forEach() является методом массива и используется для перебора всех элементов массива. В отличие от цикла for...of, он не поддерживает прерывание цикла с помощью break.

let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(num) {
    console.log(num);
});

Этот метод перебирает элементы массива и выполняет указанную функцию для каждого элемента. В данном примере будут выведены числа от 1 до 5.

7. Цикл с использованием break

Оператор break позволяет завершить выполнение цикла досрочно, даже если условие продолжает оставаться истинным.

for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;
    }
    console.log(i);
}

Этот цикл завершится, как только значение i станет равным 5, и выведет числа от 0 до 4.

8. Цикл с использованием continue

Оператор continue пропускает текущую итерацию цикла и переходит к следующей. Это полезно, когда нужно пропустить некоторые шаги в цикле при выполнении определённых условий.

for (let i = 0; i < 5; i++) {
    if (i === 2) {
        continue;
    }
    console.log(i);
}

Этот цикл пропустит итерацию, когда i равно 2, и выведет 0, 1, 3, 4.
