Числа, операции с числами, встроенные математические функции

В JavaScript все числа представлены одним типом – Number. Это 64-битное число с плавающей запятой в формате IEEE 754.

Также есть специальный тип BigInt для работы с большими целыми числами.

Безопасные числа

JavaScript использует 64-битный формат IEEE 754, но может точно представлять только целые числа в диапазоне Number.MIN_SAFE_INTEGER до Number.MAX_SAFE_INTEGER. Числа за пределами этого диапазона могут вести себя неожиданно.

ОперацияПримерРезультат
Максимальное безопасное числоNumber.MAX_SAFE_INTEGER
Минимальное безопасное числоNumber.MIN_SAFE_INTEGER
Число за пределами безопасного диапазонаNumber.MAX_SAFE_INTEGER + 1
