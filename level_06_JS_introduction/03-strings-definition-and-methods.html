Строки: определение и методы

Способы создания строк:

В JavaScript строки можно создавать несколькими способами:

  • Использование одинарных кавычек (''): 
    let str1 = 'Это строка';
  • Использование двойных кавычек (""): 
    let str2 = "Это тоже строка";
  • Использование обратных кавычек (``) — для многострочных строк и интерполяции: 
    let str3 = `Это многострочная
строка, которая может
содержать переносы строк.`;

В обратных кавычках также можно вставлять переменные:

let name = "Иван";
let str4 = `Привет, ${name}!`;

Методы строк:

Метод Описание Пример
length Возвращает длину строки. 
"Hello".length → 5
toUpperCase() Преобразует строку в верхний регистр. 
"hello".toUpperCase() → "HELLO"
toLowerCase() Преобразует строку в нижний регистр. 
"HELLO".toLowerCase() → "hello"
charAt(index) Возвращает символ на указанной позиции. 
"hello".charAt(1) → "e"
charCodeAt(index) Возвращает код символа на указанной позиции. 
"hello".charCodeAt(1) → 101
includes(substring) Проверяет, содержит ли строка подстроку. 
"hello world".includes("world") → true
startsWith(prefix) Проверяет, начинается ли строка с указанного префикса. 
"hello".startsWith("he") → true
endsWith(suffix) Проверяет, заканчивается ли строка указанным суффиксом. 
"hello".endsWith("lo") → true
indexOf(substring) Возвращает индекс первого вхождения подстроки. Если не найдено —1. 
"hello".indexOf("e") → 1
lastIndexOf(substring) Возвращает индекс последнего вхождения подстроки. Если не найдено —1. 
"hello".lastIndexOf("l") → 3
slice(start, end) Возвращает новый фрагмент строки от start до end (не включая end). 
"hello".slice(1, 4) → "ell"
substring(start, end) Возвращает новый фрагмент строки, аналогично slice, но не принимает отрицательные индексы. 
"hello".substring(1, 4) → "ell"
split(separator) Разделяет строку на массив по разделителю. 
"apple,banana,orange".split(",") → ["apple", "banana", "orange"]
replace(searchValue, newValue) Заменяет первое вхождение подстроки или регулярное выражение в строке. 
"hello world".replace("world", "everyone") → "hello everyone"
replaceAll(searchValue, newValue) Заменяет все вхождения подстроки. 
"hello hello".replaceAll("hello", "hi") → "hi hi"
trim() Удаляет пробелы с начала и конца строки. 
"  hello  ".trim() → "hello"
trimStart() Удаляет пробелы с начала строки. 
"  hello".trimStart() → "hello"
trimEnd() Удаляет пробелы с конца строки. 
"hello  ".trimEnd() → "hello"
padStart(targetLength, padString) Добавляет строки в начало, чтобы достичь заданной длины. 
"5".padStart(3, "0") → "005"
padEnd(targetLength, padString) Добавляет строки в конец, чтобы достичь заданной длины. 
"5".padEnd(3, "0") → "500"

Шаблоны строк:

Шаблоны строк в JavaScript создаются с помощью обратных кавычек (``). Они позволяют вставлять переменные и выражения непосредственно в строку, не используя конкатенацию.

Интерполяция выражений:

let name = "Иван";
let age = 30;
let greeting = `Привет, ${name}! Тебе ${age} лет.`;
console.log(greeting);  // Выведет: "Привет, Иван! Тебе 30 лет."

Многострочные строки:

let multiLine = `Это строка
с несколькими
строками.`;
console.log(multiLine);
// Выведет:
// Это строка
// с несколькими
// строками.

Шаблоны строк позволяют вставлять не только простые переменные, но и любые выражения:

let result = `2 + 3 = ${2 + 3}`;
console.log(result);  // Выведет: "2 + 3 = 5"
