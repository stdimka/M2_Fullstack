Псевдоклассы в CSS

  • :hover: a:hover { ... }

    Применяет стиль при наведении на ссылку.

    a:hover {
        color: red;
    }
  • :focus: input:focus { ... }

    Применяет стиль, когда элемент получает фокус.

    input:focus {
        border-color: blue;
    }
  • :first-child: p:first-child { ... }

    Применяет стиль к первому элементу <p> в родительском контейнере.

    p:first-child {
        font-weight: bold;
    }
  • :last-child: p:last-child { ... }

    Применяет стиль к последнему элементу <p> в родительском контейнере.

    p:last-child {
        color: green;
    }
  • :nth-child(n): li:nth-child(n) { ... }

    Применяет стиль к элементам в зависимости от значения n.

    li:nth-child(2) { background-color: yellow; } /* второй элемент */
li:nth-child(odd) { background-color: lightgray; } /* нечетные элементы */
li:nth-child(even) { background-color: lightblue; } /* четные элементы */
li:nth-child(3n) { font-weight: bold; } /* каждый 3-й элемент */
li:nth-child(3n+1) { color: red; } /* каждый 3-й элемент начиная с 1-го */
li:nth-child(-n+5) { text-decoration: underline; } /* первые 5 элементов */
  • :not(selector): p:not(.special) { ... }

    Применяет стиль ко всем <p>, кроме тех, у которых есть класс special.

    p:not(.special) {
        color: gray;
    }
