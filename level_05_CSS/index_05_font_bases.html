Основные свойства текста в CSS

1. font-family (Шрифт)

Определяет семейство шрифтов.

Times New Roman
Arial

2. font-size (Размер шрифта)

Определяет размер текста.

font-size: 12px
font-size: 20px
font-size: 32px

3. font-weight (Насыщенность шрифта)

Определяет толщину шрифта.

font-weight: normal; Обычный
font-weight: bold; Жирный
font-weight: 900; Очень жирный

4. font-style (Курсив)

Определяет стиль шрифта.

font-style: normal; Обычный
font-style: italic; Курсив
font-style: oblique; Наклонный

5. text-align left; center; right; (Выравнивание текста)

Определяет горизонтальное выравнивание текста.

Влево
Центр
Вправо

6. text-decoration (Оформление текста)

Определяет декоративные эффекты текста.

text-decoration: underline; Подчёркнутый
text-decoration: line-through; Зачёркнутый
text-decoration: overline;Надчёркнутый

7. text-transform (Преобразование регистра)

Определяет преобразование регистра букв.

text-transform: uppercase; ВЕРХНИЙ РЕГИСТР
text-transform: lowercase; нижний регистр
"text-transform: capitalize; Первая буква заглавная

8. letter-spacing (Межбуквенный интервал)

Определяет расстояние между символами.

2px
5px

9. line-height (Межстрочный интервал)

Определяет высоту строки.

1 (плотно)
1.5 (оптимально)
2 (разряжено)

10. word-spacing (Межсловный интервал)

Определяет расстояние между словами.

5px
10px
