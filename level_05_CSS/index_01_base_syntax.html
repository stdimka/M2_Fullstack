Базовый синтаксис CSS

Основные элементы CSS:

  • Правила (Rules)
  • Селекторы (Selectors)
  • Свойства (Properties)
  • Значения (Values)

Правила (Rules)

CSS состоит из правил, которые определяют стиль элементов HTML. Каждое правило содержит селектор и блок объявлений.

Пример:

p {
    color: blue;
    font-size: 16px;
}

Селекторы (Selectors)

Селекторы используются для выбора элементов HTML, к которым будут применяться стили.

  • Теговый селектор: p { ... } (Применяет стиль ко всем <p>)
  • Классовый селектор: .my-class { ... } (Применяет стиль ко всем элементам с class="my-class")
  • ID селектор: #my-id { ... } (Применяет стиль к элементу с id="my-id")
  • Атрибутный селектор: input[type="text"] { ... } (Применяет стиль на основе атрибутов элемента.)
    • [attr] — элементы, у которых есть указанный атрибут.
    • [attr="value"] — элементы с точным значением атрибута.
    • [attr^="value"] — элементы, у которых значение атрибута начинается с указанного текста.
    • [attr$="value"] — элементы, у которых значение атрибута заканчивается на указанный текст.
    • [attr*="value"] — элементы, у которых значение атрибута содержит указанный текст.

Свойства (Properties)

Свойства определяют, какие аспекты внешнего вида элемента изменяются.

  • color — цвет текста
  • font-size — размер шрифта
  • background-color — цвет фона

Значения (Values)

Значения задают конкретные параметры для свойств.

  • red — значение для color
  • 16px — значение для font-size
  • #f0f0f0 — значение для background-color

Вам необходимо:

  • запомнить стандартные свойства элементов
  • запомнить стандартные значения этих свойств
  • запомнить уникальные свойства разных элементов
  • запомнить, как эти свойства работают на практике
  • запомнить, как эти свойства влияют друг на друга.

Лучше всего это сделать через практику – писать много CSS и смотреть как он работает.
