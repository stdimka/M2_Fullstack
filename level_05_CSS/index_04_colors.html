Цвет в CSS

1. Способы задания цвета в CSS

Цвет в CSS можно задавать разными способами:

  • Название цвета (например, red, blue, green)
  • RGB (например, rgb(255, 0, 0))
  • HEX (например, #FF0000)
  • HSL (например, hsl(0, 100%, 50%))
  • RGBA и HSLA (например, rgba(255, 0, 0, 0.5))

2. Цветовая модель RGB

RGB (Red, Green, Blue) задает цвет с помощью красного, зеленого и синего компонентов.

rgb(255, 0, 0)
rgb(0, 255, 0)
rgb(0, 0, 255)

3. Цветовая модель HEX

HEX (шестнадцатеричный код) использует 6 символов для задания цвета.

#FF0000
#00FF00
#0000FF

4. Цветовая модель HSL

HSL (Hue, Saturation, Lightness) использует оттенок, насыщенность и яркость.

hsl(0, 100%, 50%)
hsl(120, 100%, 50%)
hsl(240, 100%, 50%)

5. Прозрачность: RGBA и HSLA

RGBA и HSLA добавляют альфа-канал (прозрачность) к цвету.

rgba(255, 0, 0, 0.5)
hsla(240, 100%, 50%, 0.5)
