Селекторы в CSS

Базовые селекторы

  • Теговый селектор: p { ... } (Применяет стиль ко всем <p>)
  • Классовый селектор: .my-class { ... } (Применяет стиль ко всем элементам с class="my-class")
  • ID селектор: #my-id { ... } (Применяет стиль к элементу с id="my-id")

Дополнительные селекторы

  • Универсальный селектор: * { ... } (Применяет стиль ко всем элементам)
  • Групповой селектор: h1, h2, h3 { ... } (Применяет стиль ко всем указанным тегам)
  • Дочерний селектор: div > p { color: red; }

    Применяет стиль только к непосредственным потомкам <div>.

    <div>
    <p>Этот текст станет красным.</p>
    <section>
        <p>Этот текст НЕ изменится, так как он не является ПРЯМЫМ потомком div.</p>
    </section>
</div>
  • Потомковый селектор: div p { color: green; }

    Применяет стиль ко всем вложенным <p> внутри <div>, независимо от уровня вложенности.

    <div>
    <p>Этот текст станет зелёным.</p>
    <section>
        <p>Этот текст тоже станет зелёным, так как p является потомком div.</p>
    </section>
</div>
  • Соседний селектор: h1 + p { color: blue; }

    Применяет стиль к первому <p>, который идет сразу после <h1>.

    <h1>Заголовок</h1>
<p>Этот абзац сразу после h1 и станет синим.</p>
<p>Этот абзац НЕ изменится, так как он не сразу после h1.</p>
  • Родственный селектор: h1 ~ p { color: purple; }

    Применяет стиль ко всем <p>, которые следуют за <h1> на одном уровне вложенности.

    <h1>Заголовок</h1>
<p>Этот абзац станет фиолетовым.</p>
<div>Некоторый блок</div>
<p>Этот абзац тоже станет фиолетовым, так как он является родственным элементом h1.</p>

<section>
    <p>Этот абзац НЕ изменится, так как он находится в другом родительском элементе.</p>
</section>
  • Атрибутные селекторы: input[type="text"] { ... } (Применяет стиль к input с type="text")

Псевдоклассы

  • a:hover { ... } (Срабатывает при наведении на ссылку)
  • input:focus { ... } (Срабатывает, когда input в фокусе)
  • p:first-child { ... } (Применяет стиль к первому элементу p в родительском контейнере)

Псевдоэлементы

  • p::first-line { ... } (Стилизует первую строку текста в параграфе)
  • ::before / ::after (Добавляют контент до или после элемента)
  • ::selection { ... } (Стилизует выделенный текст)
